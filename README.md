# アプリケーション名
Todoアプリ
![image](https://user-images.githubusercontent.com/68457238/210950273-fb23b599-7c8a-490f-98ef-f05b0f67c573.png)


## 作成した目的
タスクの管理のため

## 機能一覧
- Todoの一覧表示機能
- Todoの追加
- Todoの更新
- Todoの削除
- ユーザー認証
- タグ機能
- 検索機能

## 使用技術（実行環境）
- Laravel　8.83.26
